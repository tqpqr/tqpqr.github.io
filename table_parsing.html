Parsing online table directly to Pandas

using read_html from pandas.io

This is one of the easiest ways to parse online table without additional libraries.

  1. from pandas.io.html import read_html
  2. import requests
  3. from time import sleep
  4.  
  5.  
  6. # Making a list of links (for example, there are 623 pages on site, that contains a data that we are interested in)
  7. a=[]
  8. for i in range(1,624):
  9.     a.append(i)
  10.  
  11. for num in a:
  12.     url='https://www.statkod.**/usa_index'str(num)'.html'
  13.  
  14.     r=requests.get(url).text
  15.    
  16.     # Ignoring unfamiliar symbols in encoding if it is necessary.
  17.     r=r.encode('cp1251','ignore')
  18.  
  19.     # Trying to make zeros to appear in zip codes (read_html don't have 'dtype' parameter)
  20.  
  21.     tables = read_html(r, converters={'col': str})
  22.     filename='c:/temp/result'str(num)'.csv'
  23.     tables[1].to_csv(filename,sep='\t')
  24.     print (tables[1])
  25.     sleep(5)

Nice and easy.
